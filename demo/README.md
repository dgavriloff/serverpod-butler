## TL;DR

**Breakout Butler** = Zoom breakout room companion with:
- Real-time collaborative workspaces
- Live lecture transcription
- AI-powered Q&A ("What did she say about X?")
- Professor dashboard with cross-room synthesis

**Serverpod Features Used:**
- Streaming endpoints (WebSocket)
- Session messaging (Redis pub/sub)
- ORM with relations
- Protocol classes (shared models)
- Code generation
- Database migrations
- Multi-environment configs
- Built-in web server

**Demo Recommendation:** Pre-load transcript data for the 3-minute video to show AI features immediately without waiting for live transcription.
